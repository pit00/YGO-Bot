# YGO Bot
Source code of Yu-Gi-Oh! Telegram Bot write in Node.js

## Description
Fetch card image from Yu-Gi-Oh! (TCG)

## Link
For more information and how to use, access:
[@YGOh_Bot](http://telegram.me/YGOh_Bot)
